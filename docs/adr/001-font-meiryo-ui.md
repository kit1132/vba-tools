# ADR-001: 統一フォントとして Meiryo UI を採用

## ステータス

承認済み

## コンテキスト

FormatWorkbook マクロでブック全体のフォントを統一する際、どのフォントを標準とするか決定が必要だった。要件は以下の通り：

- 日本語・英語の両方で読みやすい
- Windows / macOS の両方で利用可能
- ビジネス文書に適した外観
- セル・グラフ・図形のすべてで利用可能

## 決定

統一フォントとして **Meiryo UI** を採用する。

## 結果

### ポジティブ
- Windows 標準搭載で追加インストール不要
- 日英混在テキストのバランスが良い
- UI フォントとしてコンパクトで表の視認性が高い

### ネガティブ
- macOS にはデフォルトで搭載されていない場合がある（手動インストールが必要）
- フォントが存在しない環境では `On Error Resume Next` でエラーを回避しているが、フォントは変更されない
- 印刷時に MS ゴシック等に比べて行間が若干異なる

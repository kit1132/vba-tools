# Format Workbook — 書式統一ツール

Excelブック全体の書式（フォント・表示倍率）を一括統一するVBAマクロです。

---

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `FormatWorkbook.bas` | メインマクロ（全シートのフォント＋Zoom統一） |
| `グラフや図形の配置設定をデフォルト.vb` | グラフ・図形の配置設定リセット用（未実装） |

---

## FormatWorkbook の機能

全シートに対して以下を一括適用します。

- **フォント統一**: Meiryo UI（セル・テーブル・グラフ・図形すべて）
- **表示倍率**: 80%
- **カーソル位置**: 各シートのA1セルに移動

### 適用対象の詳細

- セル全体のフォント
- テーブル（ListObject）のヘッダー・本体・集計行
- グラフのタイトル・軸ラベル・凡例・データラベル
- 図形・テキストボックスのテキスト

---

## 導入手順

1. Excelで対象ブックを開き、`Alt + F11` でVBAエディタを起動
2. **「ファイル」→「ファイルのインポート」** で `FormatWorkbook.bas` をインポート
3. `.xlsm` 形式で保存
4. `Alt + F8` → `FormatWorkbook` を選択 →「実行」

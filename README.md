# VBA シート管理ツール

Excelブック内のシートを管理するためのVBAマクロ集です。
シートが多いブック（月別・案件別など）で、目次のようにシート一覧を生成し、ワンクリックで各シートへジャンプできるようにします。

---

## マクロ一覧

このツールには3つのマクロがあります。通常は `RunAll` だけ覚えておけば問題ありません。

| マクロ名 | 機能 | 使い方 |
|---------|------|--------|
| **`RunAll`** | 下記2つをまとめて実行し、最後にシート一覧を表示。保護シートがある場合はスキップしたシート名を通知 | **通常はこれだけ実行すればOK** |
| `AddRowsAndGoHome` | 全シートの先頭に空行2行を追加し、一番左のシートへ移動。保護シートはスキップして通知 | 個別に実行したい場合のみ |
| `UpdateSheetList` | シート一覧を生成・更新 | 個別に実行したい場合のみ |

---

## 各マクロの詳細

### RunAll（一括実行）

以下の順序で2つのマクロをまとめて実行します。

```
1. AddRowsAndGoHome → 全シートの先頭に空行を追加
2. UpdateSheetList   → シート一覧を生成・更新
```

- 実行完了後、**シート一覧が表示された状態**になります
- 保護されたシートがあった場合、完了後にスキップしたシート名をダイアログで通知します
- 処理中にエラーが発生した場合、エラー内容をダイアログで表示します

### AddRowsAndGoHome（空行追加）

全シートの先頭に2行の空行を追加し、一番左のシートへ移動します。

- 「シート一覧」シートは**対象外**（スキップ）
- **保護されたシート**もスキップし、完了後にスキップしたシート名を通知

### UpdateSheetList（シート一覧の生成）

ブック内の全シート名を一覧にした「シート一覧」シートを作成します。

- ブックの**一番左**に「シート一覧」シートを配置
- 各シート名に**ハイパーリンク**が付き、クリックでそのシートへジャンプ

**イメージ：**

```
┌──────────────────────────────┐
│ シート一覧                    │
├──────────────────────────────┤
│  シート名          ← ヘッダー（太字）
│  1月売上           ← クリックで「1月売上」シートへ
│  2月売上           ← クリックで「2月売上」シートへ
│  3月売上           ← クリックで「3月売上」シートへ
│  顧客マスタ        ← クリックで「顧客マスタ」シートへ
└──────────────────────────────┘
```

## 導入手順

### Step 1：VBAエディタを開く

Excelで対象のブックを開き、`Alt + F11` を押してVBAエディタを起動します。

### Step 2：SheetManager.bas を取り込む

1. VBAエディタのメニューから **「ファイル」→「ファイルのインポート」** を選択
2. `SheetManager.bas` を選んでインポート

### Step 3：ThisWorkbook に Option Explicit を記入する

1. VBAエディタ左側のプロジェクトエクスプローラーで **「ThisWorkbook」** をダブルクリック
2. コード画面の先頭に `Option Explicit` と記入する（変数の宣言漏れを防止するための設定です）

### Step 4：マクロ有効ブックとして保存

**「ファイル」→「名前を付けて保存」** でファイル形式を **「Excel マクロ有効ブック (.xlsm)」** にして保存します。

---

## 使い方

### マクロの実行方法

`Alt + F8` → マクロ名を選択 →「実行」

通常は **`RunAll`** を選択すれば、空行追加とシート一覧更新がまとめて実行されます。

---

## 注意事項

- シートの追加・削除・名前変更を行った場合は、`RunAll` を手動で実行してシート一覧を更新してください。
- ブックを `.xlsx` 形式で保存するとマクロが削除されます。必ず **`.xlsm`** 形式で保存してください。

# VBA シート管理ツール

Excelブック内のシートを管理するためのVBAマクロ集です。
シートが多いブック（月別・案件別など）で、目次のようにシート一覧を生成し、ワンクリックで各シートへジャンプできるようにします。

---

## マクロ一覧

| マクロ名 | 機能 | いつ使うか |
|---------|------|-----------|
| **`UpdateSheetList`** | シート一覧を生成・更新し、シート一覧を表示する | **シートの追加・削除・名前変更をしたとき** |
| `AddRows` | 全シートの先頭に空行2行を追加し、シート一覧に戻る | 各シートにヘッダー用の空行が必要なときだけ |

> **注意**: `AddRows` は実行するたびに2行ずつ増えます。必要なとき以外は実行しないでください。

---

## 各マクロの詳細

### UpdateSheetList（シート一覧の生成・更新）

ブック内の全シート名を一覧にした「シート一覧」シートを作成します。

- ブックの**一番左**に「シート一覧」シートを配置
- 各シート名に**ハイパーリンク**が付き、クリックでそのシートへジャンプ
- 実行完了後、**シート一覧が表示された状態**になります

**イメージ：**

```
┌──────────────────────────────┐
│ シート一覧                    │
├──────────────────────────────┤
│  シート名          ← ヘッダー（太字）
│  1月売上           ← クリックで「1月売上」シートへ
│  2月売上           ← クリックで「2月売上」シートへ
│  3月売上           ← クリックで「3月売上」シートへ
│  顧客マスタ        ← クリックで「顧客マスタ」シートへ
└──────────────────────────────┘
```

### AddRows（空行追加）

全シートの先頭に2行の空行を追加し、シート一覧に戻ります。

- 「シート一覧」シートは**対象外**（スキップ）
- **保護されたシート**もスキップされます（通知なし）

## 導入手順

### Step 1：VBAエディタを開く

Excelで対象のブックを開き、`Alt + F11` を押してVBAエディタを起動します。

### Step 2：SheetManager.bas を取り込む

1. VBAエディタのメニューから **「ファイル」→「ファイルのインポート」** を選択
2. `SheetManager.bas` を選んでインポート

### Step 3：ThisWorkbook に Option Explicit を記入する

1. VBAエディタ左側のプロジェクトエクスプローラーで **「ThisWorkbook」** をダブルクリック
2. コード画面の先頭に `Option Explicit` と記入する（変数の宣言漏れを防止するための設定です）

### Step 4：マクロ有効ブックとして保存

**「ファイル」→「名前を付けて保存」** でファイル形式を **「Excel マクロ有効ブック (.xlsm)」** にして保存します。

---

## 使い方

### マクロの実行方法

`Alt + F8` → マクロ名を選択 →「実行」

### よくある操作

| やりたいこと | 実行するマクロ |
|-------------|--------------|
| シートを追加・削除・名前変更した後に一覧を更新したい | `UpdateSheetList` |
| 全シートにヘッダー用の空行を追加したい | `AddRows` |

---

## 注意事項

- シートの追加・削除・名前変更を行った場合は、`UpdateSheetList` を手動で実行してシート一覧を更新してください。
- `AddRows` は実行するたびに空行が2行ずつ追加されます。重複実行にご注意ください。
- ブックを `.xlsx` 形式で保存するとマクロが削除されます。必ず **`.xlsm`** 形式で保存してください。

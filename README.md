# VBA シート管理ツール

Excelブック内のシートを管理するためのVBAマクロ集です。
シートが多いブック（月別・案件別など）で、**目次のようにシート一覧を自動生成**し、ワンクリックで各シートへジャンプできるようにします。

---

## できること

### 1. シート一覧の自動生成（`UpdateSheetList`）

ブック内の全シート名を一覧にした「シート一覧」シートを自動作成します。

- ブックの**一番左**に「シート一覧」シートを配置
- 各シート名に**ハイパーリンク**が付き、クリックでそのシートへジャンプ
- 新しいシートを追加すると**自動で一覧を更新**
- シートを削除した場合も**自動で一覧を更新**

**イメージ：**

```
┌──────────────────────────────┐
│ シート一覧（自動生成）        │
├──────────────────────────────┤
│  シート名          ← ヘッダー（太字）
│  1月売上           ← クリックで「1月売上」シートへ
│  2月売上           ← クリックで「2月売上」シートへ
│  3月売上           ← クリックで「3月売上」シートへ
│  顧客マスタ        ← クリックで「顧客マスタ」シートへ
└──────────────────────────────┘
```

### 2. 全シートへの空行一括追加（`AddRowsAndGoHome`）

全シートの先頭に2行の空行をまとめて追加します。

- 「シート一覧」シートは**対象外**（自動でスキップ）
- **保護されたシート**も自動でスキップし、完了後にスキップしたシート名を通知
- 処理後、一番左のシートに自動で移動

## ファイル構成

```
vba-tools/
├── README.md            ← このファイル
├── ThisWorkbook.cls     ← イベント処理（シート追加・削除の検知）
└── SheetManager.bas     ← マクロ本体（シート一覧生成・空行追加）
```

| ファイル | 役割 |
|---------|------|
| `ThisWorkbook.cls` | シートの追加・削除を自動検知し、シート一覧を更新する |
| `SheetManager.bas` | `UpdateSheetList` と `AddRowsAndGoHome` の2つのマクロを格納 |

## 導入手順

### Step 1：VBAエディタを開く

Excelで対象のブックを開き、`Alt + F11` を押してVBAエディタを起動します。

### Step 2：SheetManager.bas を取り込む

1. VBAエディタのメニューから **「ファイル」→「ファイルのインポート」** を選択
2. `SheetManager.bas` を選んでインポート

### Step 3：ThisWorkbook にコードを貼り付ける

1. VBAエディタ左側のプロジェクトエクスプローラーで **「ThisWorkbook」** をダブルクリック
2. `ThisWorkbook.cls` の中身（`Option Explicit` 以下すべて）をコピーして貼り付け

### Step 4：マクロ有効ブックとして保存

**「ファイル」→「名前を付けて保存」** でファイル形式を **「Excel マクロ有効ブック (.xlsm)」** にして保存します。

---

## 使い方

### シート一覧を手動で更新する

`Alt + F8` →「UpdateSheetList」を選択 →「実行」

> 通常はシートの追加・削除時に自動で更新されるため、手動実行が必要な場面は**シート名を変更したとき**のみです。

### 全シートに空行を追加する

`Alt + F8` →「AddRowsAndGoHome」を選択 →「実行」

---

## 注意事項

- シート名を変更した場合は、一覧への自動反映はされません。`UpdateSheetList` を手動で実行してください。
- ブックを `.xlsx` 形式で保存するとマクロが削除されます。必ず **`.xlsm`** 形式で保存してください。
